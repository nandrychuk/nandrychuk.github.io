Serviceability Checks
Provincial Check ⓘ Verifies that the address is in NS, NL, or PEI.
Serviceable | Province does not make the address "Not Serviceable".
Not Serviceable | Address is not serviceable.
—
Service Area Check ⓘ Checks if the address is in a "high serviceability likelyhood" area.
Serviceable | Area does not make the address "Not Serviceable".
Unknown Serviceable | Not within an area but could still be serviceable.
Not Serviceable | Address is not serviceable.
—
City Check ⓘ Checks if the address is in a known serviceable city/town.
Serviceable | City does not make the address "Not Serviceable".
Unknown Serviceable | Not within a city/town explicitly marked as serivceable or not serviceable.
Not Serviceable | Address is not serviceable.
—
Weird Building Check ⓘ Checks if the address is a known "weird building".
Serviceable | Weird building list does not include the address.
Not Serviceable | Address is not serviceable because it is in the Weird Buildings list.
